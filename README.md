# 🧮 Calculator

A free, all-in-one calculator website built with HTML, CSS, and JavaScript.

## ✨ Features

- **Standard Calculator** — Basic arithmetic with history
- **Scientific Calculator** — Trig, logarithms, powers, constants
- **Finance Calculator** — EMI, SIP, Compound Interest, GST
- **Unit Converter** — Length, Weight, Temperature, Area, Volume, Speed, Time, Data, Energy
- **Date & Age Calculator** — Age, date difference, BMI
